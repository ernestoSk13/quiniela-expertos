## Cambios técnicos por área

### Sistema de puntos
- `ScoringConfig.tsx`: eliminados campos `exactScore` y `exactKnockoutWithTie`; renombrado `correctResult → correctPrediction`; banner "Modo resultado simple" en el formulario
- `firestoreConfig.ts`: interfaz `ScoringConfig` actualizada (4 campos, alineada con Cloud Functions)
- El usuario cambió el valor de `correctPrediction` de 3 → 1 pt directamente desde el admin UI en producción

### Onboarding
- `Onboarding.tsx`: flujo de 4 pasos — Perfil(1) → Demo(2) → Bonus(3) → Acceso(4); `totalSteps` 3/4 según PWA
- `StepDemo.tsx` (nuevo): partido ficticio MEX vs USA con `ResultPicker` reutilizado; reveal 2-1 con animación pop; `+1 pt ✓` o `+0 pts ✗`
- `StepInstall.tsx`: reemplazado por instrucciones de bookmark (Ctrl+D, ⭐ Chrome, Compartir → Añadir marcador iOS); ilustración ribbon bookmark SVG
- `StepProfile.tsx`: avatar rectangular 96×128px (portrait, estilo Panini); botones "Tomar foto" (`capture="user"`) y "Galería"; instrucción "Agrega tu avatar"

### Admin — Panel de premios
- `AdminPremios.tsx` (nuevo): `/admin/premios`; formulario con selector de jugador, título, checkboxes puntos/posición, swatches de 6 colores
- `PaniniCard.tsx` (nuevo): tarjeta 340×480px; gradiente de acento cubre toda la tarjeta; avatar 132×170px con borde blanco; textos blancos contrastantes; compatible con html2canvas

### Navegación admin/jugador
- `AdminLayout.tsx`: botón "← Ver como jugador" en sidebar footer (desktop) y "Jugador" en header mobile
- `Dashboard.tsx`: botón "Admin" en header, visible solo si `user.role === 'admin'`

### Zonas horarias
- `User.timezone?: string` (IANA)
- `useUserTimezone.ts` (nuevo): lee `user.timezone` o fallback browser

## Decisiones de diseño

| Decisión | Razonamiento |
|----------|--------------|
| Puntos cambiados a 1pt (desde admin UI, no código) | Más simple; el admin lo modificó directamente desde `/admin/config` en producción |
| Fase 13 cancelada definitivamente | Sin marcadores exactos no hay narrativa visual para el slideshow |
| Avatar rectangular portrait en onboarding | Consistencia con las tarjetas Panini; más espacio para la foto |
| Modo claro pausado | Inline styles `rgba(255,255,255,X)` en componentes impiden override solo desde CSS |
| `fix-timestamps` script con dry-run por defecto | Protección para operaciones destructivas en producción; 5s de gracia antes de escribir |
| Dropdown para ThemeSelector | 14 banderas en fila saturaban la navbar |
| `ThemeContext` con `THEMES.forEach` en lugar de lista hardcodeada | Elimina bug donde los temas nuevos no se limpiaban al cambiar |
